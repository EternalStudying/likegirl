## 已做决策
| 决策 | 理由 |
|------|------|
| 将三个规划文件放在项目根目录 | `planning-with-files-zh` 要求规划文件放在项目目录 |
| 当前只记录上下文，不创建实现计划 | 用户明确说明还没有指派任务 |
| 以后只处理前端范围 | 用户指定我是前端 Agent |
| 首页导航上移使用 CSS 变量公式 | 让导航中心跟随雪坡高度变化，而不是固定单一像素值 |
| 雪坡改由 CSS 伪元素生成 | 用户要求去掉雪坡图片，改用 CSS 构建抛物线曲线 |
| 不处理移动端 | 用户明确要求不要考虑移动端 |
| 将首页封面与导航提升为登录后公共布局 | 用户要求点击导航其他页面时，新页面出现在导航栏下方，而不是整体出现新页面 |
| 规划文件按价值记录，不按每次修改强制记录 | 用户要求由我判断本次修改是否值得记录，值得则记录，否则不记录 |
